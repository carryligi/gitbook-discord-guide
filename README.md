# Keyword Pinger Guide

The **Keyword Pinger** monitors specific channels and notifies you instantly via DM when your keywords are found. It integrates directly with **Silently** to autostart your tasks — even when you're away.

{% hint style="info" %}
**Heads up:** All notifications are sent via **Direct Message**. You must be a server member to use the bot.
{% endhint %}
