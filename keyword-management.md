# Keyword Management

Manage what the bot searches for. You can set keywords for all categories or limit them to specific channels.

| Command | Description |
|---------|-------------|
| `/add_keyword keyword: "your_keyword" channel: "IDs"` | Adds a keyword for specific channels. Separate multiple keywords with a comma (e.g., `dunk,jordan,yeezy`). Restrictions apply to all words. |
| `/remove_keyword "keyword"` | Removes a specific keyword from your list. |
| `/active_keywords` | Displays all currently monitored keywords. |
