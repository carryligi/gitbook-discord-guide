# Best Usage Workflow

Follow these steps for the optimal setup:

1. **Setup** — Start with `/add_keyword` (use commas for multiple words!).
2. **Integrate** — Use `/silently key` and `/silently_activate` to link your bot.
3. **Refine** — Set filters like `/autostart_min_stock` or a schedule.
4. **Safety** — Set a cooldown with `/pinger_cooldown 5`.
5. **Go!** — Enable monitoring with `/activate_pinger` and wait for DMs!
