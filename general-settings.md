# General Settings & Controls

Control the main bot functions and view your configuration.

| Command | Description |
|---------|-------------|
| `/activate_pinger` | Globally turns monitoring **on** for your user. |
| `/deactivate_pinger` | Globally turns monitoring **off** for your user. |
| `/pinger_cooldown "minutes"` | Sets a cooldown per channel and keyword. Example: If Channel A pings "oasis", it goes on a 5-min cooldown for that keyword — you will still receive pings for that keyword from Channel B! |
| `/keywordpinger_settings` | Shows a complete overview of all configured settings (Keywords, Cooldowns, Silently, Pushover, etc.). |
